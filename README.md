# Weekend Wizard

## Overview
Weekend Wizard is a Python application. This project contains the following main files:
- `agent_fun.py`
- `server_fun.py`

## Requirements
- Python 3.x

## Installation
1. Clone or download this repository to your local machine.
2. Ensure Python 3 is installed on your system.

## Execution Instructions

### To run the agent:
```
python agent_fun.py
```

### To run the server:
```
python server_fun.py
```
